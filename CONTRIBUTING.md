## Working in the code

### Environment

Tested with Node 8 and NPM 5 and up.

### Setup

Clone the repo and run an install:

```shell
$ git clone git@github.com:dhleong/ps4-waker.git && cd ps4-waker && npm install
```

### Run tests

To run all tests execute

```shell
$ npm test
```
